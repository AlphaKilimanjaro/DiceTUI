
class TUI: TUI for terminal dice game

TUI() create and set up TUI

setDice(): set die 1 and 2

showDice(): show dice on screen (terminal)

requestRoll(int player): request roll from player 1 or 2

welcome(): write welcome on screen (terminal)

theEnd(): write end game message to screen (terminal)
